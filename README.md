# LangChain Playlist 🚀

This repository contains my **LangChain learning playlist**, where I explore and implement the core components of LangChain along with practical examples.

The goal of this repository is to understand how to build **LLM-powered applications** using LangChain and related tools.

## 📚 What this repository covers

This playlist will cover the **6 core components of LangChain**:

1. **Models** – Working with LLMs and chat models
2. **Prompts** – Prompt templates and prompt engineering
3. **Chains / Runnables** – Creating pipelines using LangChain runnables
4. **Memory** – Adding conversational memory to applications
5. **Retrieval** – Using vector databases for information retrieval
6. **Agents** – Building autonomous agents that can use tools

## 🧠 Retrieval Augmented Generation (RAG)

This repository will also include **RAG implementations**, covering:

* Document loading
* Text splitting
* Embeddings generation
* Vector databases (FAISS / ChromaDB)
* Retrieval pipelines
* RAG based Q&A systems
